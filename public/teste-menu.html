🎯 Como acessar o menu:

1. Abra o app em http://localhost:3004/
2. Procure o botão GRANDE e VERDE no canto superior esquerdo
3. O botão tem o texto "MENU" e um ícone de três linhas (☰)
4. Clique no botão para abrir o menu lateral
5. O menu desliza da esquerda com todas as opções

🛠️ Soluções Comuns
Problema: Botão do Menu não aparece

Solução:

  1. Abra o Console do navegador (F12)
  2. Procure por erros em vermelho
  3. Recarregue a página (Ctrl + F5)
  4. Limpe o cache do navegador
Problema: Menu não abre ao clicar

Solução:

  1. Verifique se o JavaScript está habilitado
  2. Teste em modo anônimo (Ctrl + Shift + N)
  3. Tente outro navegador (Chrome, Edge, Firefox)
  4. Execute o teste de diagnóstico acima
